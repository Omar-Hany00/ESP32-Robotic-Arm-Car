# ESP32 Robotic Arm Car

A custom ESP32-based robotic arm car designed for embedded control, wireless operation, and integrated PCB-based hardware management.

The project combines robotics, embedded systems, PCB design, and motor actuation into a compact mobile robotic platform featuring a servo-actuated robotic arm and custom-designed electronics.

---

## Features

- ESP32-based embedded control system
- Custom PCB designed using KiCad
- DC motor drive system
- Servo-controlled robotic arm
- Integrated power and signal routing
- Compact mobile robotic platform

---

## Hardware Overview

The system consists of:

- ESP32 microcontroller
- Custom-designed PCB
- DC drive motors
- Servo motors for robotic arm actuation
- Embedded power distribution circuitry

---

## Hardware Design

The PCB and schematic were designed using KiCad to integrate:
- ESP32 communication and control
- Motor driver connections
- Servo actuation outputs
- Power management and routing

---

## Tools Used

- KiCad
- ESP32
- Embedded C / Arduino Framework
- DC Motors
- Servo Motors
